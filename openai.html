監測資料查詢彙整

選用GPT-4o模型，利用Langchain框架，打造Agent資料檢索系統。讓使用者能透過問答方式，查詢關聯式資料庫資料， 並結合Prompt engineering技術，回答內容可以自訂選擇為文字、折線圖或柱狀圖。 (因屬機敏資料，故不顯示實際數字)

Agent平均分數 : 95%

  • 程式語言 : Python
  • 使用框架 : Langchain
  • 開發系統 : Ubuntu
